Calculations

Do calculations only on issues that are in "Done" column category.

Execution Time = Σ issue time spent in Active columns
Delay Time = Σ issue time spent in Delay columns

Cycle Time = Execution Time + Delay Time
Process Efficiency = Execution Time / Cycle Time
A transactional process is considered Lean when the execution time is less than one third of the delay time, or about 25 percent of the cycle time.
